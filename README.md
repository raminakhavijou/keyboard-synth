A contemporary web-based synthesizer that turns your computer keyboard into a musical instrument.

## Features

- **Dual Timbre System**: Sustained pads and quick pluck sounds
- **Polyphonic**: Play multiple notes simultaneously  
- **40 Keys Mapped**: Numbers, QWERTY, ASDF, and ZXCV rows
- **Mouse & Touch Support**: Works on desktop and mobile devices
- **Contemporary Sound Design**: Detuned oscillators, filtered waveforms

## Quick Start
```bash
npm install
npm run dev
```

Open http://localhost:5173

## How to Play

- **Numbers (1-0)**: Sustained pad sounds
- **QWERTY Row**: Q-Y are plucks, U-P are pads
- **ASDF Row**: Quick pluck sounds  
- **ZXCV Row**: Bass notes

## Technology

React + Vite + Web Audio API + Tailwind CSS
